# 一、BIT 介绍与合规性

## 1. BIT 介绍

BIT 的前身是 Matrixport，2019 年在新加坡成立，创始人是吴忌寒，公司已经有七年历史了。

这家公司之前主要做的是加密货币相关的业务，不过定位并非币安欧易这种纯交易所，更多是一个数字金融服务平台，提供交易托管、借贷、结构化理财、双币理财、资产托管这些金融服务。

公司实力方面，BIT 累计融资超过了 1.67 亿美元，投资方包括 DST Global、Tiger Global、Polychain 这些一线机构，估值超过 10 亿美元，算得上是金融独角兽公司。

目前平台管理资产规模超过 60 亿美元，月交易量超过 70 亿美元，入选了 2024 年胡润全球独角兽榜。

上个月的时候，Matrixport 正式更名为 BIT，全称是 Bridge Into Tomorrow，我猜改名的一方面原因，可能也是为了合规上市做准备。

而我们今天要讲的核心内容，是 BIT 最近几个月正式官宣的新功能，美股交易。

## 3. 美股交易的合规架构

我觉得这部分应该是大家最关心的地方，涉及到了资产安全的问题，我来详细拆解一下 BIT 美股交易背后的合规架构。

BIT的美股业务由自有的持牌主体MatrixGelephu Pte Ltd支持，再由美国的持牌上手券商提供清算与托管，

上手券商受SEC监管，是DTC、NSCC 及 OCC成员机构，用户资产受到SIPC的保障。

整个架构分三层。

最上面一层是清算托管券商，也就是美国持牌券商。

BIT平台上所有的美股交易，最终都是通过在美国持牌的券商在美国SEC监管体系内完成的。

中间一层是 BIT 的不丹持牌主体 Matrix Gelephu Pte Ltd。这个实体以 Introducing Broker 的身份为用户提供服务。

你可以理解为它是一个介绍经纪商，负责把用户的交易指令传到上手券商去执行。

最底层就是我们普通用户。

![](/content-assets/bit-broker-matrixport-open-account/02-jpg.webp)

用户的钱是否能够得到保障，主要看资产隔离，即使平台破产倒闭，我们用户的钱还是能够拿回来，这是最关键的。

BIT 平台的用户资产是独立于公司运营资金托管的。买的股票真实持有，托管在美国受监管的券商体系下，受到 SIPC 保障。

根据美国法律，这些资产是客户代管资产，即便券商破产也不会进入清算池。我们的资产受到法律强制保护。

所以说，在这家平台投资交易美股，资金安全还是挺有保障的，已经是传统美股券商的保障力度了。

## 4. 和传统券商的区别

传统券商，比如长桥或者盈透，需要有海外地址证明或者身份证明才可以正常开户，需要你有境外银行账户才能入金。如果想用 USDT 入金，需要提供各种资产证明，要求非常严格。

而 BIT 都不需要，身份证可以直接开户。我们既可以用境外银行账户入金，也可以用 USDT 或者 USDC 直接入金，7x24 小时秒级到账。

但本质上，你在 BIT 上买的美股和在传统券商上买的美股没有区别。都是真实持有，享有股东权益，包括股息分红和投票权。

我想这也是 BIT 美股交易区别于美股代币的一大特点。

而且 BIT 目前的美股交易功能，基本可以对标传统券商，包括盘前盘中盘后交易、碎股交易、支持交易个股 + 各类  等。

只要成功完成开户，就能领到 100 USDT 的平台费抵扣券以及 50% 年化的理财加息券。

# 四、BIT 手续费率

我们再简单总结一下 BIT 美股手续费率，和长桥的收费标准基本一致。

**交易佣金：免费。**

**平台费：** 固定收费 0.005 USD/股，每笔订单最低收取 0.99 USD。

**交易活动费用：** 0.000195 USD/股，每次成交最低 0.01 USD，最高 9.79 USD。这个费用只在卖出股票时收取。

**证券交易委员会费用：** 交易金额 × 0.0000206，每次成交最低 0.01 USD。同样只在卖出时收取。

**综合审计跟踪费用：** 0.000053 USD/股。

# 五、写在最后

目前大陆用户想投资交易美股个股/ETF，主流的方式有两种。

第一种，传统合规券商，比如长桥、盈透。合规性没话说，资产安全有保障，功能也很完善。但问题是开户门槛高，需要海外地址证明或者身份证明，入金必须依赖境外银行账户。对于很多没有港卡或者海外银行的用户来说，第一步就卡住了。

第二种，美股代币。开户门槛确实低，主流交易所都支持。但现阶段持有的并不是真实股票，没有股东权益。本质上只是一个锚定股价的代币资产。而且在合规监管层面，目前也存在一定的不确定性。

BIT 找到了一个很巧妙的平衡位置。

它把开户门槛降到了最低，大陆身份证直接开户，不需要海外地址或者身份证明。入金门槛也非常低，USDT 直接充值，不需要境外银行账户。

但它也没有牺牲合规性。背后有美国持牌券商做交易清算，用户持有的是真实股票，受 SIPC 保障，资产独立托管，整个链路都在合规框架下运行。

如果你手里有 USDT，想要投资美股，但又不想折腾开港卡、办海外银行这些流程，现在确实可以试试 BIT。
