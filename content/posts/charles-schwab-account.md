大陆用户开通美股券商账户，有两个最大的痛点，一个是需要海外地址，另一个是担心 CRS 税务问题。

今天我要给大家分享的嘉信理财，这两个痛点都不存在，开户不需要海外地址，而且不会受到 CRS 的影响。

想要投资美股的朋友可以试试这家券商。

# 一、什么是嘉信理财？

我们先快速介绍一下嘉信。

它是一家在纽交所上市的美国本土券商。

成立于 1971 年，到今年已经有 50 多年历史了，比我们最常听到的 1977 年成立的盈透券商，还要早一些。

嘉信目前是美国最大的零售券商，管理的客户资产总规模接近 12 万亿美元，是全世界排得上号的头部金融机构。

接下来，我再给大家说说嘉信理财的一些优势。

1️⃣ 免佣金

交易购买股票或者 ETF，没有任何手续费，也没有最低金额要求，这一点比盈透还要划算一些。

2️⃣ 安全性

嘉信是美国本土老牌券商，受到美国 SEC 的监管，每个嘉信账户也都受到 SPIC 的保护，有几十万美元的保额。

即使嘉信券商破产倒闭，我们的资产也是安全的。

所有合规券商在用户资产安全层面，都大差不差。

3️⃣ 免 CRS

CRS 是经济合作与发展组织，也就是 OECD，主导的全球税务信息自动交换系统。简单说，如果你在香港、新加坡、开曼这些地方开了金融账户，你的账户信息会被自动上报回内地税务局。

而嘉信是美国本土券商，美国走的是 FATCA 体系，只管美国公民的海外账户，没有加入 CRS 自动交换。这意味着你在嘉信的大陆账户，账户信息不会被自动上报。

不过这里我必须提醒大家一点，免除 CRS 只是现行规定下的事实，并不等同于不需要交税。

4️⃣ 无需海外地址

嘉信开户可以直接使用大陆的地址证明，不需要提供海外地址或者身份，非常方便。

之前嘉信开户还是有一些门槛的，需要入金两万五千美元，现在已经全部取消了，开户方面可以说是没有任何门槛。

当然，嘉信也有自己的劣势，目前我体验下来，最大的感受就是嘉信的 UI 非常差，用户体验很糟糕。

在比烂方面，嘉信相比起盈透，可以说是有过之而无不及，大家开户之后去用一用就知道了。

我自己使用下来，感觉用户体验还不如盈透。

我想很多美国的年轻用户，大概率不会用嘉信这种历史悠久的券商，而是选择 Robinhood 这类新券商。

除此之外，嘉信的开户流程非常繁琐，从提交开户申请，到开户通过，中间可能需要一个月的时间，而且还可能会踩坑。

还有就是，嘉信的出入金流程，相比起盈透，也会更麻烦一些。

# 二、开户前的准备清单

在开户之前，有两样东西需要提前准备好。

一个是护照或者身份证。嘉信作为美国券商，优先识别护照，我开户的时候就是用护照作为申请材料的。

有护照首选护照。如果没有护照的话，也可以用身份证，嘉信官方是接受的。

另外一个就是国内的地址证明，银行对账单，或者水电煤账单。嘉信不要求海外地址，大陆地址就可以。

除此之外，我也建议大家最好在电脑上操作开户，整个过程还是挺复杂的，我当时自己开，用了差不多二三十分钟的时间。

# 三、从 0 到 1 开户流程

接下来，我就从 0 到 1，给大家演示一下嘉信券商的具体开户流程。

首先，打开嘉信理财官网地址 https://international.schwab.com/ 

这个是嘉信的国际版入口，专门给非美国居民用的，它的前缀有一个international，如果没有这个前缀的话，是美国本土居民专用的入口，一定要区分清楚。

打开之后，点击右上角的 Open Account 按钮。

接着，它这里会提示需要准备的材料，和我们刚才说的一致。

![](/content-assets/charles-schwab-account/02-png.webp)

往下拉，需要选择一下国家地区，我们直接选择 China。

![](/content-assets/charles-schwab-account/03-png.webp)

选择完之后，就会跳转到下个网页，然后我们往下滑动。

这里显示出了三个账户，我们需要开的是 Individual Accont，没有任何开户金额的要求，点击 continue。

![](/content-assets/charles-schwab-account/04-png.webp)

进入这个开户界面之后，它会告诉你一些基本的注意事项，比如说开户没有任何资金要求，让你准备好地址证明以及身份证件。

![](/content-assets/charles-schwab-account/05-png.webp)

拉到底部，我们点击 Get started。

![](/content-assets/charles-schwab-account/06-png.webp)

![](/content-assets/charles-schwab-account/07-png.webp)

接着，就是填写个人的基本资料，都得用英文填写，不能用中文。

从这一步开始，建议大家一次性填好，然后直接提交申请，不要隔几个小时或者几天之后再回来填写，可能会遇到问题。

First Name 就填写你护照上或者身份证上的名，比如你叫王大锤，这里就填写大锤的大写英文。

Middle Name 大家一般都没有，Last Name 就填写你护照上或者身份证上的姓，比如你叫王大锤，这里就填写王的大写英文。

下面的 suffix 以及known by another name，一般不需要填写。

![](/content-assets/charles-schwab-account/08-png.webp)

国籍选择 China，Date of birth，就填写你证件上的出生日期，用护照申请，就填写护照上的，用身份证申请，就填写身份证上的，保持一致。

电子邮件和手机号就正常填写，如果你有国外手机号的话，最好用国外的，+86容易收不到验证码。

如果你开完户之后，+86手机号收不到验证码，登陆不上去，最后我也给出了相应解决方案。

当然，长期解决方案还是开通一个国外手机号，大家可以参考一下我之前出的这期 eSIM 从 0 到 1 教程 [Xesim + Club SIM 海外手机号教程](https://invest-nav.com/tutorials/overseas-sim/text/xesim-club-sim/)

选择护照的话，需要填写过期时间以及证件的发行国家。

接下来，填写一下邮箱收到的验证码，点击continue。

此外，还有一个常见的问题需要着重说一下。如果你开户时候填写的是 +86 手机号，那么在登陆嘉信理财账户的时候，可能会遇到收不到验证码的问题。

大家遇到这个问题的话，直接拨打 +1 415 667 8400 这个嘉信国际热线，拨打进去之后会让你选择中文服务，然后会问你当前是否完成开户。你就选择没有开户，然后就会自动转接到中文客服服务。

走到人工客服之后，你就把登录不上的问题告诉客服，然后按照客服的指示，一步步操作就可以了。

嘉信理财的客服服务还是挺不错的，电话很快就能拨通，即使是周末，也有相应的人工服务。
