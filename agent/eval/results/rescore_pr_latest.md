### 检索层（本次新算）

对每条 Top-K 结果，若命中任一 `gold_tags` 或 `gold_section_keywords`，记为**相关段**。

| 指标 | 定义 |
|------|------|
| **Precision@K** | 相关段数量 / 实际返回条数 |
| **Recall@K** | 被命中的金标「方面」(每个 tag/kw 算一个方面) / 金标方面总数 |
| **Hit@K** | 是否至少 1 条相关段（0/1） |
| **Hit Rate** | 各题 Hit@K 的平均（有金标的题） |
